# かんたんモーションメーカー

透過PNGのキャラ画像を1枚読み込んで、軽い「生きてる感じ」のモーションを付けた
スプライトシートを作る、自分専用のローカルツールです。
AI動画生成やボーン・物理は使わず、Canvasの拡縮/移動/部分回転だけで作ります。

## 起動方法

サーバ不要。**`index.html` をダブルクリック**してブラウザで開くだけです。

> プロジェクトのプレビューサーバ（`npx serve tools`）経由で開く場合は、
> `http://localhost:<port>/motion-maker/`（末尾スラッシュ付き）で開いてください。
> スラッシュが無いと相対パスがずれて読み込めません。

## 使い方（6ステップ）

1. **画像を読み込む** … 上の大きいボタン、または画面へドラッグ＆ドロップ
2. **動きを選ぶ** … 呼吸 / ふわふわ / ぷるぷる / しっぽ揺れ / 翼パタ / まばたき
3. **場所を選ぶ**（しっぽ・翼・目のときだけ）… 「範囲を選ぶ」→ 動かしたい所をドラッグで囲む
   - 赤い点が回転の支点（付け根）です。範囲は体に少しかぶせ気味に囲うと自然です。
4. **強さを調整** … スライダーで動きの大きさを変える
5. **再生で確認** … 「▶ プレビュー再生」でループ再生（速さも調整可）
6. **保存** … 「スプライトシートを保存」（横並び1枚）か「各コマを個別PNGで保存」

書き出しは**背景透明のまま**。市松模様はプレビュー表示用で、保存画像には入りません。

## 各モーションのしくみ（ざっくり）

| モーション | 範囲 | 中身 |
|---|---|---|
| 呼吸 | 不要 | 足元を固定して縦に伸び縮み |
| ふわふわ | 不要 | 全体を上下に平行移動 |
| ぷるぷる | 不要 | 横に伸びると縦が縮む揺れ（スライム向け） |
| しっぽ揺れ | しっぽ | 範囲を切り抜き、付け根を支点に左右回転 |
| 翼パタ | 翼 | 同上＋軽い折りたたみ感 |
| まばたき | 目 | 範囲に肌色のまぶたを被せて1回閉じる |

## ファイル構成

```
motion-maker/
├─ index.html      … 画面（左プレビュー / 右設定）
├─ css/style.css   … 配色・レイアウト
└─ js/
   ├─ motions.js   … ★モーション定義（ここに足せば動きが増える）
   ├─ frames.js    … 切り抜き/穴/支点/肌色の下ごしらえ＋フレーム生成
   ├─ player.js    … 再生/停止ループ
   ├─ exporter.js  … スプライトシート/個別PNG 書き出し
   └─ app.js       … UI配線・画像読込・範囲ドラッグ・全体制御
```

## 新しい動きの足し方

`js/motions.js` の `list` にオブジェクトを1個足すだけで、UIのボタンも自動で増えます。

```js
{
  id: 'shake', name: 'ぶるぶる（小刻み）', needsRegion: false,
  defaults: { frames: 6, strength: 40, fps: 16 },
  apply(dctx, fc, t, s) {
    const dx = Math.sin(t * Math.PI * 2 * 2) * s * 4; // 速めの横揺れ
    drawWhole(dctx, fc, { dx });   // ※共通ヘルパは motions.js 内に定義済み
  }
}
```
